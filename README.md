# Food101 Image Classification using EfficientNetB0 - Fine-Tuning

## Overview

Built a deep learning image classification system on the Food101 dataset using TensorFlow and Keras. The project leverages transfer learning with EfficientNetB0, mixed precision training (FP16), and fine-tuning to improve classification performance while reducing training time.

## Key Features

* Transfer Learning using EfficientNetB0
* Fine-Tuning of pretrained model layers
* Mixed Precision Training (float16)
* TensorBoard experiment tracking
* Model Checkpointing
* Food101 Dataset (101 food categories)
* End-to-End Deep Learning Pipeline

## Tech Stack

* Python
* TensorFlow
* Keras
* TensorFlow Datasets
* NumPy
* Matplotlib

## Workflow

1. Load Food101 dataset
2. Data preprocessing and resizing
3. Feature extraction using EfficientNetB0
4. Train classification head
5. Fine-tune pretrained layers
6. Evaluate model performance
7. Save and deploy trained model
